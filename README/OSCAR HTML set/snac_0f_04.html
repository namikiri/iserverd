SNAC(0f,04)  CLI_INTERESTS_REQ 

Client use this to request interests list from server. Server should reply via SNAC(0f,05).

 00 0f   word   SNAC family
 00 04   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 empty (no snac data)

Example SNAC dump with flap header: 

  2a 02 e1 ca 00 0a 00 0f 00 04 00 00 99 d0 07 62  *..............b
