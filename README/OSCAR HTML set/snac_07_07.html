SNAC(07,07)  SRV_ACCOUNTxCONFIRMxREPLY 

This is the reply to the SNAC(07,06). It contain status flag. You'll receive email with instructions about removing TRIAL flag if status=0x00. Also you may receive status=0x1E if account allready confirmed or status=0x23 if server can't start confirmation procedure.

 00 07   word   SNAC family
 00 06   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
   xx xx   word   confirmation status
 00 04   word   TLV.Type(0x04) - url
 xx xx   word   TLV.Length
 xx ..   string   url (http://www.aol.com)

Example SNAC dump with flap header: 

  2A 02 12 8B 00 22 00 07 00 07 00 00 00 01 00 06  *...."..........
  00 1E 00 04 00 12 68 74 74 70 3A 2F 2F 77 77 77  ......http://www
  2E 61 6F 6C 2E 63 6F 6D                          .aol.com
