SNAC(15,03)/07DA/0302  META_REGISTRATION_STATS_ACK 

Result for client SNAC(15,02)/07D0/0AA5 registration stats report. If success byte equal 0x0A - operation was finished succesfully, if not - database error.

 00 15   word   SNAC family
 00 03   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 00 01   word   TLV.Type(1) - encapsulated META_DATA
 xx xx   word   TLV.Length
 xx xx   word (LE)   data chunk size (TLV.Length-2)
 xx xx xx xx   dword (LE)   request owner uin
 DA 07   word (LE)   data type: META_DATA
 02 00   word (LE)   request sequence number
 02 03   word (LE)   data subtype: META_REGSTATS_ACK
 0A   char   success byte

  00 01 00 0d 0b 00 bc 2b d0 0e da 07 12 00 02 03  .......+........
  0a                                               .
