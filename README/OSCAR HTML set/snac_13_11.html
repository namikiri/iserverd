SNAC(13,11)  CLI_SSI_EDIT_BEGIN 

Use this before server side information (SSI) modification. Also you should send SNAC(13,12) after SSI modification. You could also use "import" transaction mode to add contacts requiring authorization. Just add 0x00010000 to snac data to start import transaction. See also snac list for SSI service here.

 00 13   word   SNAC family
 00 11   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 empty (or 0x00010000 for import transaction)

Example SNAC dump with flap header: 

  2A 02 22 96 00 0A 00 13  00 11 00 00 00 00 00 11 *.".............
