SNAC(85,03) [IServerd extension] SRV_BROADCAST_REPLY 

This is the server reply for client broadcast message SNAC(85,02). This family currently supported only by IServerd. Result flag: 0x000a - message was sent successfully, 0x0032 - can't send message

 00 85   word   SNAC family
 00 03   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx   word   Result flag

Example SNAC dump with flap header: 

  2A 02 97 63 00 0C 00 85 00 03 00 00 00 00 00 14  *.×c............
  00 0a                                            ..
