SNAC(03,0C)  SRV_USER_OFFLINE  

Server send this when user from your contact list goes offline. See also additional information about online userinfo block. Complete snac list for this service here.

 00 03   word   SNAC family
 00 0C   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 Following user info may be repeated more then once
 xx   char   uin string length
 xx ..   string   uin string
 xx xx   word   warning level (unused in ICQ)
 xx xx   word   Number of TLV in info-tlvlist
 00 01   word   TLV.Type(0x01) - user class
 xx xx   word   TLV.Length
 xx xx xx xx   dword   User class
