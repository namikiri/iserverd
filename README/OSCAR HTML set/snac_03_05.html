SNAC(03,05)  CLI_BUDDYLIST_REMOVE  

Use this this to remove buddies from your client-side contact list during session. You can add buddies to contact list via SNAC(03,04). See also complete snac list for this service here.

 xx xx xx xx   dword   SNAC request-id
 xx   byte   buddy id #1 strlen
 xx ..   string   buddy id #1 string
 ...    ...    ...
 xx   byte   buddy id #n strlen
 xx ..   string   buddy id #n string
