SNAC(13,09)  CLI_SSIxUPDATE 

This can be used to modify either the name or additional data for any items that are already in your server-stored information. It is most commonly used after adding or removing a buddy: you should either add or remove the buddy ID# from the type 0x00c9 TLV in the additional data of the parent group, and then send this SNAC containing the modified data. Server should reply via SNAC(13,0E).

 00 13   word   SNAC family
 00 08   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx ..   struct   list of items

Example SNAC dump with flap header: 

  2A 02 7C C2 00 27 00 13 00 09 00 00 00 07 00 09  *.|..'..........
  00 07 42 75 64 64 69 65 73 72 CD 00 00 00 01 00  ..Buddiesr......
  0C 00 C8 00 08 0F 63 78 2A 63 6F 5A 9B           ......cx*coZ.
