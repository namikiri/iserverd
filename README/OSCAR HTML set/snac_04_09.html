SNAC(04,09)  SRV_ICBM_WARNINGxACK 

This is the server ack for SNAC(04,08) client send warning command. See also SNAC(04,01) and SNAC(04,05) for more information.

 00 04   word   SNAC family
 00 09   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx   word   warning level gain
 xx xx   word   new warning level

Example SNAC dump with flap header: 

  2A 02 D5 B8 00 0E 00 04 00 09 00 00 00 17 00 08  *...............
  00 C8 01 ED                                      ....
