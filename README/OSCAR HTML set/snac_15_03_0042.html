SNAC(15,03)/0042  SRV_END_OF_OFFLINE_MSGS  

This is the last SNAC in server response to cli_offline_msgs_req SNAC(15,02)/003C. It doesn't contain message - it is only end_of_sequence marker.

 00 15   word   SNAC family
 00 03   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 00 01   word   TLV.Type(1) - encapsulated META_DATA
 00 0B   word   TLV.Length
 09 00   word (LE)   data chunk size (TLV.Length-2)
 xx xx xx xx   dword (LE)   msg target uin
 42 00   word (LE)   data type: end of offline messages
 02 00   word (LE)   request sequence number
 
 xx   char   dropped messages flag
