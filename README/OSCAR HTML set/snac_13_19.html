SNAC(13,19)  SRV_SSI_AUTHxREQUEST 

This is the authorization request from another client because it wants to add you to its contact list. You may just ignore this request or you could reply via SNAC(13,1A) - authorization reply.

 00 13   word   SNAC family
 00 19   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx   byte   length of the uin string
 xx ..   string   uin string (ascii)
 xx xx   word   length of the reason string
 xx xx   word   reason (message) string
 00 00   word   unknown

Example SNAC dump with flap header: 
              
  2A 02 2C 3A 00 1C 00 13  00 19 00 00 00 01 00 19 *...............
  07 36 32 31 38 38 39 35  00 06 70 6C 65 61 73 65 .6218895..please
  00 00                                            ..
