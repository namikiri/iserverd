SNAC(04,0C)  SRV_MSG_ACK  

This is the server ack for client message sent via SNAC(04,06). Server send it only if client requested it via empty TLV(3) in message snac. This ack mean that server accepted message for delivery.

 00 04   word   SNAC family
 00 0C   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx xx xx
 xx xx xx xx		   msgid   client message id cookie (8 bytes)
 00 02   word   message channel
 xx   char   sender uin string len
 xx ..   string   sender uin string
