SNAC(0D,04)  CLI_ROOMxINFOxREQ 

Room info request.

 00 0D   word   SNAC family
 00 04   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
  Following chat room identification info may be repeated more then once
 xx xx   word   exchange
 xx   byte   cookie length
 xx ..   array   cookie
 xx xx   word   instance number
 xx   byte   detail level
