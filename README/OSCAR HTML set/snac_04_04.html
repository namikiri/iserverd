SNAC(04,04)  CLI_ICBM_PARAM_REQ  

Use this snac to request your icbm parameters from server. Server should reply via SNAC(04,05). You can change them using SNAC(04,02).

 00 04   word   SNAC family
 00 04   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 empty (no snac data)        

Example SNAC dump with flap header: 

  2A 02 33 6F 00 0A 00 04  00 04 00 00 00 00 00 04 *.3o............
