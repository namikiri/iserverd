SNAC(0D,02)  CLI_CHATNAV_RIGHTS_REQ 

Client use this SNAC to request chat navigation service parameters and limitations. Server should reply via SNAC(0D,09) - chat navigation information reply.

 00 09   word   SNAC family
 00 02   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 empty (no snac data)

Example SNAC dump with flap header: 

  2A 02 22 9B 00 0A 00 0D 00 02 00 00 00 00 00 02 *.".............
