SNAC(01,17)  CLI_FAMILIES_VERSIONS  

This is the client response to SNAC(01,03). This snac contain families versions which client want's from server. Server should respond with SNAC(01,18). Client sends this SNAC during protocol negotiation sequence. See also known families list.

 00 01   word   SNAC family
 00 17   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx   word   family number #1
 xx xx   word   family version
 ...    ...    ...
 xx xx   word   family number #n
 xx xx   word   family version
