SNAC(13,1C)  SRV_SSI_YOUxWERExADDED 

Server send this snac to clients, that announced the use of family 0x13 in SNAC(01,17). This is the "you-were-added" message meaning that somebody (snac contain his/her screenname) added you to his/her roster.

 00 13   word   SNAC family
 00 1C   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx   byte   length of the uin string
 xx ..   string   uin string (ascii)

Example SNAC dump with flap header: 
                                                       
  2A 02 3D 67 00 12 00 13  00 1C 00 00 00 01 00 1C *.=g.............
  07 36 32 31 38 38 39 35                          .6218895
