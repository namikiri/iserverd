SNAC(01,0D)  SRV_RESUME  

After this SNAC client, paused by SNAC(01,0B) may continue send packets to BOS. Migration sequence used to redirect client to new BOS server during current BOS shutdown. Families array is optional.

 00 01   word   SNAC family
 00 0D   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx   word   family number #1
 ...    ...    ...
 xx xx   word   family number #n

Example SNAC dump with flap header: 

 2A 02 E5 65 00 0A 00 01  00 0D 00 00 8E CC C4 DE *..e............
