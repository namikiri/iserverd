SNAC(15,02)/07D0/040B  META_SET_EMAIL_USERINFO 

Client use this to change extended email user information. Server should reply via SNAC(15,03)/07DA/0087 which contain flag byte. If flag byte doesn't equal 0x0A - there was an error saving info.

 00 01   word   TLV.Type(1) - encapsulated META_DATA
 xx xx   word   TLV.Length
