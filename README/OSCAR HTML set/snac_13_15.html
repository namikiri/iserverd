SNAC(13,15)  SRV_SSI_FUTURExAUTHxGRANTED 

You'll receive this when somebody grants future authorization to you. You can use SNAC(13,14) to send such authorization grant.

 00 13   word   SNAC family
 00 15   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx   byte   length of the uin string
 xx ..   string   uin string (ascii)
 xx xx   word   length of the reason string
 xx xx   word   reason (message) string
 00 00   word   unknown

Example SNAC dump with flap header: 

  2A 02 1D 56 00 1A 00 13  00 15 00 00 99 D0 07 62  *..V...........b
  07 34 36 37 38 38 31 32  00 04 73 75 72 65 00 00  .4678812..sure..
