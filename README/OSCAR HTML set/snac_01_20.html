SNAC(01,20)  CLI_VERIFICATION_REPLY  

This is the client verification reply for server client-verification SNAC(01,1F). It contain MD5 hash (16 bytes long) of requested by server winaim.exe static memory region.

 00 01   word   SNAC family
 00 20   word   SNAC subtype
 00 00   word   SNAC flags
 xx xx xx xx   dword   SNAC request-id
 xx xx xx xx   array   MD5 hash array (16 bytes)
 xx xx xx xx  
 xx xx xx xx  
 xx xx xx xx
